Err plugin for weather reporting in Japan
=========================================

Installation
------------

.. code-block:: bash

   !repos install https://github.com/attakei/errbot-jp-weather.git

Usage
-----

.. code-block:: bash

   > !jpweather_area
   ...
   東京都
   ...

   > !jpweather_city 東京都
   (not working...)
   ...
   !jpweather_city 東京都
   ...

   > !jpweather 東京
   東京の天気: 今日は雨
